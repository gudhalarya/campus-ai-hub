AetherCampus • Final Deployment Guide

AetherCampus Final Runbook

Local-first AI platform with Rust backend + containerized local models and optional cloud fallback.

1) Quick Start

./setup

This auto-detects machine specs, picks best runtime mode, pulls model tier(s), writes runtime config, and starts containers.

2) Access URLs

  • Frontend: http://localhost:8080
  • Health: http://localhost:8000/health
  • Ready: http://localhost:8000/ready
  • Metrics: http://localhost:8000/metrics

3) Setup Modes

./setup --auto
./setup --local
./setup --cloud
./setup --no-start

4) Local Model Tiers

  • Low: qwen2.5:1.5b, phi3:mini, llama3.2:1b
  • Mid: qwen2.5:3b, llama3.2:3b, gemma2:2b
  • High: qwen2.5:7b, llama3.1:8b, gemma2:9b

Runtime routing tiers: LOCAL_MODEL_FAST, LOCAL_MODEL_BALANCED, LOCAL_MODEL_QUALITY.

5) Backend Endpoints

  • POST /api/chat (streaming)
  • GET /api/utility/templates
  • POST /api/utility/generate
  • GET /api/ai/report

6) Operations

docker compose -f infra/docker-compose.runtime.yml logs -f

docker compose -f infra/docker-compose.runtime.yml down
docker compose -f infra/docker-compose.runtime.yml --profile local up -d --build
